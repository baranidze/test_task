# test_task
Test task for CodeIT

Test task for CodeIT

The code is written on PHP version 7.0.10. 
File "task.sql" contains SQL-code to create DB and fill it with countries. 
File "Db.php" contains the code that allows you to connect to the database. 
File "clearFunction.php" contains function that clears a string of all unnecessary characters. 
This function is used when you sign in and sign up. The main page is "index.php". 
It contains sign in form and registration link. File "registration.php" contains registration form. 
There are fields that need to be filled. File "page.php" contains user information. 
It displays user's name and email. User login that is sent in URL (method GET) is needed to get user information from DB.

The solution to this task took two and a half hours.
